Steps to get started with ptyhon selenium package on Windows OS
1. install python3 (everything selected on installation page)
2. install selenium package using pip3
   a. open cmd in admin mode
   b. pip3 install selenium
3. Download latest chrome driver from 'http://chromedriver.chromium.org/downloads'
   The term 'chrome driver' is used for Chrome browser. 
   In short, chrome driver simulates the behaviour chrome browser and used to test web stuffs
4. Place dowloaded 'chromedriver.exe' at 'C:\chromedriver\' location (you can put anywhere)
5. Add 'C:\chromedriver\' location to 'Path' environment variable
6. Run you python program :-)
